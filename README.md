# afyaboratz-site — Simple Landing Template

Files:
- `index.html` — main page
- `styles.css` — styling
- `logo.svg` — placeholder logo

Quick start:
1. Open `index.html` in a browser.
2. Edit text in `index.html` and styles in `styles.css` to match your brand.
3. Replace `logo.svg` with your logo (same filename) for automatic header update.

Deploy:
- Push to a GitHub repo and enable GitHub Pages (branch: main or gh-pages) to publish.
- Or upload the files to any static host (Netlify, Vercel, Surge).

Notes:
- Contact form is a demo; set a real `action` or wire to an API/mailing service.
- Colors and fonts are in `styles.css` root variables for easy theming.
